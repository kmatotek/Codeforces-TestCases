# Problem Description

Boneca Ambalabu gives you a sequence ofnintegersa1,a2,…,an.
Output the maximum value of(ak⊕a1)+(ak⊕a2)+…+(ak⊕an)among all1≤k≤n. Note that⊕denotes thebitwise XOR operation.

## Input
The first line contains an integert(1≤t≤104) – the number of independent test cases.
The first line of each test case contains an integern(1≤n≤2⋅105) – the length of the array.
The second line of each test case containsnintegersa1,a2,…,an(0≤ai<230).
It is guaranteed that the sum ofnover all test cases does not exceed2⋅105.

## Output
For each test case, output the maximum value on a new line.
